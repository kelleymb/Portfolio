Born To Read

Reading is one of my favorite pastimes. Born To Read is a community for fellow book lovers who would like the opportunity to rate and share what they've read recently!

Technologies Used (Front End): React, React Router, HTML5, CSS3 (scratch, no frameworks), Enzyme (testing), && Jest (testing). Deployed via Vercel.

Technologies Used (Back End): Node.js, Express.js, Passport/Passport-local Authentication, Bcrypt, RESTful API, Postgres, Knex.js (SQL Query Builder), Postgrator CLI, Mocha (testing), Chai (testing), && Supertest (testing). Deployed via Heroku.

Bottoms Up!

In the age of quarantine, nightlife and the bar scene have seemingly ceased to exist. But fearnot, with Bottoms Up! you can still stay home, stay safe, and be your own bartender! Users can search and post drink recipes, as well as use the random drink generator (courtesy of Cocktail DB API) and search for local breweries (courtesy of Open Brewery DB API).

Technologies Used (Front End): React, React Router, HTML5, CSS3 (scratch, no frameworks), Enzyme (testing), && Jest (testing). Deployed via Vercel.

Technologies Used (Back End): Node.js, Express.js, RESTful API, Postgres, Knex.js (SQL Query Builder), Postgrator CLI, Mocha (testing), Chai (testing), && Supertest (testing). Deployed via Heroku.
